1. Platform Overview

Melitech CRM is an enterprise-grade, multi-tenant Customer Relationship Management platform built for organisations operating across East Africa and beyond. It unifies client management, invoicing, accounting, human resources, procurement, project management, communications, and business intelligence into a single, cohesive web application.

4. CRM — Client Management

The CRM module is the core of client relationship tracking. It provides a comprehensive client registry with full contact details, activity history, associated invoices, projects, and customisable fields.

Features

  • Client Directory — Searchable, paginated list with bulk operations and CSV/PDF export
  • Client Profiles — Detailed view with tabs for contacts, invoices, projects, payments, activities
  • Custom Fields — 14+ field types (text, number, date, dropdown, checkbox, etc.) per entity
  • Advanced Filtering — Dynamic filters with saved filter presets

API Architecture

All API endpoints use tRPC v11 with end-to-end TypeScript type safety. The API supports:

  • 108 routers covering all modules
  • Zod v4 input validation on every procedure
  • JWT Bearer authentication on all protected endpoints
  • Pagination with cursor-based and offset-based support
  • Batch requests via tRPC batching
